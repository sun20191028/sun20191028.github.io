ConcurrentLinkedQueue

无界非阻塞线性安全队列，尾部进头部出，采用的cas算法，保证其线性安全，头尾都是采用慢更新的方式，所以获取的头尾，并非真正的头部和尾部

  1. 因为无界，所以add 和 offer 是一样的。
  2. 当 队列为空的时候，poll和 peek 都返回null
  3. remove 相当于 poll 加了返回值非空校验，element也相当于 peek加了返回值非空校验poll和peek为AbstractQueue的方法，继承该类都具有此特征

LinkedBlockingQueue

采用加锁的方式保证其线性安全
默认大小为 Integer.MAX_VALUE
